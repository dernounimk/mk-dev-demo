# 🚀 MK DEV – Personal Portfolio & Services Website

A modern, responsive portfolio website for **MK DEV** (a multi-talented computer science student at Mohamed Khider University in Biskra, Algeria). The site showcases front-end development skills, services offered (Front-end, UI/UX, Hosting & Domain), portfolio projects, pricing plans, and a contact section.

🌐 **Live Demo**: [dernounimk.github.io/mk-dev-demo](https://dernounimk.github.io/mk-dev-demo/)

---

## ✨ Key Features

- **Hero Section** – Introduction as a "multi-talented student" with a focus on front-end programming.
- **Services Section** – Four main services: **Front-end**, **UI/UX**, **Hosting & Domain**, **Guarantee** (maintenance & minor modifications).
- **Design Qualities** – Highlights responsive design, modern & clean design, clean code, and browser friendliness.
- **Portfolio Gallery** – Filterable work samples (All, App, Photography, Web).
- **About / Stats** – Years of experience (5), completed projects (84), awards received (4), and mails sent (824).
- **Client Evaluation** – Placeholder for testimonials / assessments.
- **Skills Section** – Visual indicators for Adobe, HTML & CSS, JavaScript, Python (based on opinions/assessments).
- **Video Section** – Placeholder for "MK DEV AWESOME VIDEO HERE".
- **Pricing Plans** – Four hosting plans (Basic $19, Premium $29, Pro $39, Platinum $49) with feature lists.
- **Contact Section** – Email subscription form for news.
- **Fully Responsive** – Works on all devices.
